## Description
気象庁から提供されているAPIを利用してCLIから天気情報を取得するアプリケーションです

## Usage
hoge
```
tenkiGetter [オプション] <場所>

オプション
-h --help ヘルプを表示
-v --version　バージョンを表示
-d --day　短期天気概況の取得
-w --week　週間天気概況の取得
```
